** Massey triple product
 If [α]⬝[β] = 0 ⇒ α⬝β = dσ.
 If [β]⬝[γ] = 0 ⇒ β⬝γ = dτ.
 Then define the triple product with
 ⟨α, β, γ⟩ = [σ⬝γ - α⬝τ]
